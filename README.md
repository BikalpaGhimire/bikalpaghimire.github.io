# Personal Website

This is my personal academic website built with Jekyll and the [al-folio](https://github.com/alshedivat/al-folio) theme.

## About

This website showcases my research, publications, projects, and professional activities.

## Credits

The structure and design of this website are inspired by [Angus F. Chapman's website](https://afchapman.github.io/). See [CREDITS.md](CREDITS.md) for full attribution.

## Theme

This site uses the **al-folio** Jekyll theme, a beautiful theme designed for academics and researchers.

## Features
- **Modern Aesthetics**: Glassmorphism navbar, Roboto typography, and card hover effects.
- **Dark Mode**: Automatic dark mode support based on system preference (defaulting to dark).
- **Responsive**: Fully responsive design for mobile and desktop.
- **Academic Focused**: Pre-built layouts for publications, research projects, and CV.

## Local Development

To run this site locally:

1. Install Ruby and Jekyll
2. Clone this repository
3. Run `bundle install`
4. Run `bundle exec jekyll serve`
5. Visit `http://127.0.0.1:4000` in your browser
